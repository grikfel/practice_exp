# practice_exp
Данные к экплуатационной практике. Разработка нейронной сети, выявляющая тему текста

# Ключевые результаты

- Точность на тестовой выборке: 94.67%
- Датасет: AG News (127,600 новостных статей)
- Модель: DistilBERT (дистиллированная версия BERT)
- Количество классов: 4 (World, Sports, Business, Sci/Tech)
- Фреймворк: PyTorch + Hugging Face Transformers

# Цель проекта

Разработка и обучение нейронной сети для автоматического определения тематики текстовых документов, что позволяет:
- Автоматизировать рубрикацию новостного контента
- Анализировать медиапотоки в реальном времени
- Персонализировать новостные ленты пользователей
